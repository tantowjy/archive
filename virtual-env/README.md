# Virtual Environment

## Python

### Windows 
-   Show the installed Python version
    ```sh
    py --list
    ```
-   Create Python virtual environment
    ```sh
    py -3.x -m venv <name_env>
    ```
-   Activate virtual environment
    ```sh
    .\<name_env>\Scripts\activate
    ```
-   Deactivate virtual environment
    ```sh
    deactivate
    ```
